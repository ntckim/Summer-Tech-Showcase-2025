Breadth-First Search (BFS)

Problem Description

Implement a Breadth-First Search algorithm to traverse a graph. BFS explores all neighbors at the current depth before moving to nodes at the next depth level, making it ideal for finding shortest paths in unweighted graphs.

How BFS Works

Breadth-First Search works by exploring all nodes at the current level before moving to the next level. Think of it like ripples in a pond - you explore outward layer by layer from the starting point.

Key Characteristics

  • Uses a queue: Processes nodes in first-in-first-out order
  • Level-order traversal: Visits all nodes at depth d before visiting nodes at depth d+1
  • Shortest path: Guarantees shortest path in unweighted graphs
  • Complete search: Will find a solution if one exists

Pseudocode

BFS(graph, start):
    queue = new Queue()
    visited = new Set()
    
    queue.enqueue(start)
    visited.add(start)
    
    while queue is not empty:
        vertex = queue.dequeue()
        print vertex
        
        for each neighbor in graph[vertex]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.enqueue(neighbor)

Time & Space Complexity

  • Time Complexity: O(V + E) where V = vertices, E = edges
  • Space Complexity: O(V) for the queue and visited set
