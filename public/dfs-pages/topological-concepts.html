Topological Sort

Problem Description

Order the vertices of a Directed Acyclic Graph (DAG) such that for every directed edge (u, v), vertex u comes before vertex v in the ordering. This is useful for scheduling tasks with dependencies.

How Topological Sort Works

The algorithm can use either DFS (with finishing times) or Kahn's algorithm (using in-degrees). Both approaches ensure that dependencies are respected in the final ordering.

Key Characteristics

  • DAG requirement: Only works on Directed Acyclic Graphs
  • Dependency resolution: Respects precedence constraints
  • Multiple solutions: May have more than one valid ordering
  • Linear ordering: Produces a linear sequence of vertices

Pseudocode

TopologicalSort-DFS(graph):
    visited = new Set()
    stack = new Stack()
    
    for each vertex v in graph:
        if v not in visited:
            DFS-TopSort(graph, v, visited, stack)
    
    result = empty list
    while stack is not empty:
        result.add(stack.pop())
    
    return result

DFS-TopSort(graph, vertex, visited, stack):
    visited.add(vertex)
    
    for each neighbor in graph[vertex]:
        if neighbor not in visited:
            DFS-TopSort(graph, neighbor, visited, stack)
    
    stack.push(vertex)  // Add to stack after processing all neighbors

TopologicalSort-Kahn(graph):
    inDegree = new Array(|V|, 0)
    queue = new Queue()
    result = empty list
    
    // Calculate in-degrees
    for each vertex u in graph:
        for each neighbor v in graph[u]:
            inDegree[v]++
    
    // Add vertices with 0 in-degree to queue
    for each vertex v in graph:
        if inDegree[v] == 0:
            queue.enqueue(v)
    
    while queue is not empty:
        vertex = queue.dequeue()
        result.add(vertex)
        
        // Reduce in-degree of neighbors
        for each neighbor in graph[vertex]:
            inDegree[neighbor]--
            if inDegree[neighbor] == 0:
                queue.enqueue(neighbor)
    
    // Check for cycle
    if result.size() != |V|:
        return "Graph has cycle - topological sort not possible"
    
    return result

Time & Space Complexity

  • Time Complexity: O(V + E) for both DFS and Kahn's methods
  • Space Complexity: O(V) for recursion stack or queue
