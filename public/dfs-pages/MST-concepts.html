Minimum Spanning Tree (MST)

Problem Description

Find a subset of edges in a connected, undirected graph that connects all vertices with the minimum total edge weight and forms a tree (no cycles).

How MST Works

A Minimum Spanning Tree connects all vertices in a graph using the smallest possible total edge weight. It contains exactly V-1 edges for V vertices and has no cycles.

Key Characteristics

  • Connects all vertices: Every vertex is reachable from every other vertex
  • Minimum weight: Has the smallest possible total edge weight
  • No cycles: Forms a tree structure with V-1 edges
  • Unique (usually): MST is unique if all edge weights are distinct

Common Applications

  • Network design (telecommunications, transportation)
  • Circuit design and electrical networks
  • Cluster analysis in machine learning
  • Approximation algorithms for traveling salesman
  • Image segmentation and computer vision

Algorithms

Two main algorithms for finding MST:

  • Kruskal's Algorithm: Edge-based, uses Union-Find
  • Prim's Algorithm: Vertex-based, uses priority queue
