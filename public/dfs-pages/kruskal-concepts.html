Kruskal's Algorithm (MST)

Problem Description

Find the Minimum Spanning Tree (MST) of a connected, undirected graph. A spanning tree connects all vertices with the minimum total edge weight and no cycles.

How Kruskal's Works

The algorithm sorts all edges by weight and greedily adds the smallest weight edge that doesn't create a cycle. It uses Union-Find data structure to efficiently detect cycles.

Key Characteristics

  • Greedy approach: Always selects minimum weight edge
  • Edge-based: Considers edges in sorted order
  • Union-Find: Uses disjoint set data structure
  • Cycle avoidance: Ensures no cycles in spanning tree

Time & Space Complexity

  • Time Complexity: O(E log E) for sorting edges
  • Space Complexity: O(V) for Union-Find structure
